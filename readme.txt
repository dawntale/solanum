== Description ==
Solanum lets you build any kind of website from a personal blog, travel, food, to portfolio website for showcasing your works. It uses Bootstrap 4 grid as its base so it's fully responsive and it has 1080p screen support. This theme support customizer which allow you to change the background color, forground color, header image, logo, and many more. You can also choose the sidebar position whether right or left.

== Frequently Asked Questions ==

= Can I change the position of sidebar? =

Yes, you can. Go to customizer and you will find theme layouts setting. Then you can choose the sidebar position either on right or left.

= How to disable the sidebar? =

It's pretty simple. Just remove all the widget on sidebar.

= Does it support JetPack Infinite Scroll? =

It supports all JetPack modules.
